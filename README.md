# Telegram Subscription Bot

A production-ready Telegram bot that supports paid subscriptions using Stripe and provides private invite links to users.

## Features (MVP)
- Stripe monthly subscription (test mode)
- Private invite link after payment
- Admin controls to manage users and subscriptions
- Force join subscription channel

## Tech Stack
- Python (Pyrogram)
- MongoDB
- Stripe
- Docker
- Railway
